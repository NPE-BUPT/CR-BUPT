## 前言
lab 项目致力于打造一个完整的高校实验室预约系统，采用现阶段流行技术实现。

##  项目介绍
lab 是一套实验室预约系统，包括前台预约系统和后台管理系统。基于SpringBoot+Mybatis实现。前台预约系统包含实验室查询、预约、取消预约、预约记录查询、公告通知、仪器设备操作配置说明等模块。后台管理包括实验室管理、预约管理、用户管理、公告管理等模块
